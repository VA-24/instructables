

- Creating an archive of a folder 
```
tar -czvf archive.tar.gz stuff
```

- Creating an archive of a file 
```
tar -czvf archive.tar.gz stuff
```
- Extracting tar file 
```
tar -xzvf archive.tar.gz
```
